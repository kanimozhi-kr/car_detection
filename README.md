
# Car Insurance Fraud Detection App

This is a Streamlit app for detecting fraudulent car insurance claims using a PyTorch model.

## Files
- `app.py`: Streamlit application code
- `fraud_detector.pth`: Your trained PyTorch model (upload this to the repo)
- `requirements.txt`: Python dependencies

## How to Deploy on Streamlit Cloud
1. Push this repository to GitHub.
2. Go to [Streamlit Cloud](https://streamlit.io/cloud) and sign in.
3. Click **Deploy an app** and select your GitHub repo.
4. Add `requirements.txt` for dependencies.
5. Deploy and get your public URL.

## Local Testing
```bash
pip install -r requirements.txt
streamlit run app.py
```
